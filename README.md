# 🏛️ Blockchain Land & Property Registration System

A production-grade, government-level land registry built on **Hyperledger Fabric**.

## 🔥 What Makes This Different
- **Permissioned blockchain** — not public Ethereum, actual enterprise-grade Hyperledger Fabric
- **3-channel design** — public records, private bank mortgage data, internal government approvals
- **SBT Ownership Certificates** — non-transferable Soulbound Tokens (can't be stolen/sold on OpenSea)
- **AI Fraud Detection** — ML model flags suspicious transfers before they hit the ledger
- **Dispute Resolution Module** — smart contract handles court stay orders with 90-day arbitration

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Blockchain | Hyperledger Fabric 2.5 |
| Smart Contracts | Go 1.21 (chaincode) |
| Backend | Node.js + Express |
| Frontend | Next.js 14 |
| AI Module | Python + FastAPI |
| Storage | IPFS via Pinata |
| Database | PostgreSQL 16 + PostGIS |
